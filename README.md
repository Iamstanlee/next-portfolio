
# Portfolio

V4 of my Portfolio site built using `Typescript`, `Nextjs`, and `Styled-components`

## Run Locally

Clone the project

```bash
  git clone https://github.com/iamstanlee/next-portfolio
```

Go to the project directory

```bash
  cd next-portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
